# Welcome to Better Scientific Software

This tutorial will be presented on Tuesday 10 November 2020 as part of the [Supercomputing 2020](https://sc20.supercomputing.org) (SC20) conference.  [Registration](https://sc20.supercomputing.org/attend/register/) for the tutorial program ("TUT") will be required to participate.
